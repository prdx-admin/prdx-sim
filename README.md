# Paradox Ecosystem Simulator

Моделирование децентрализованной экономической системы с токеном PRDX.

## Быстрый старт

```bash
pip install -r requirements.txt
jupyter notebook paradox_model.ipynb

## Описание

Симулятор агентов с разными психологическими профилями в экономике:

- Mint/Burn операции с комиссиями
- Майнинг с отрицательным EV
- 9 типов агентов (умные, агрессивные, консервативные)

## Результаты

Автоматически генерируются графики:

- Динамика цены и резервов
- ROI по профилям агентов
- Статистика майнинга
Распределение средств
